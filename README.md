# WebAppBank – Bank Portal System (Demo)

A demo web application simulating a **banking information portal with an internal intranet system**.

The project represents a simplified banking organization website that publishes informational articles and announcements, alongside a restricted internal administration panel used for content management.

This project was created for educational and portfolio purposes to demonstrate full-stack web development using ASP.NET, separation of system layers, and basic CMS-like architecture.

---

## Public Website (Bank Portal)

The public-facing part of the system serves as a banking information portal for articles and announcements.

### Features:
- Display of banking news and articles
- Article listing page
- Article details view
- Simple navigation system
- Clean public-facing UI

---

## Intranet System (Admin Panel)

The internal system is designed for managing content published on the public portal.

### Features:
- Create, edit, and delete articles
- Content management dashboard

---

## Technology Stack

- ASP.NET (C#)
- MVC / Razor Pages architecture
- HTML5, CSS3, JavaScript
- Visual Studio solution structure

---

## Key Concepts Demonstrated

- Separation of public portal and internal intranet system
- CMS-like architecture (content management system)
- Backend-driven web application design
- Basic authentication and access control
- Clean project organization in ASP.NET
